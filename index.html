Our projects

AngouriMath

AngouriMath is a computer algebra library for basic and complex expression manipulation, be that arithmetic, calculus, or printing in LaTeX, parsing from string, and many more. It might be convenient to use in industrial spheres thanks to such features as the ability to interrupt a computation.
Website · GitHub

MxEngine

MxEngine is an educational modern-C++ general-purpose 3D game engine. Right now MxEngine is developed only by me, #Momo, but any contributions are welcome and will be reviewed. Fow now MxEngine supports OpenGL as graphic API and targeting x64 only.
GitHub

GenericTensor

It is the only fully-implemented generic-tensor library for C#. Allows to work with tensors of custom types.
GitHub

MonoBind

Monobind is a lightweight header-only library that exposes C++ types in C# and vice versa, mainly to create C# bindings of existing C++ code.
GitHub

DotnetBenchmarks

In this repo we are collecting benchmarks for different situations, as well as some thoughts and articles for developers about improving their software's performance.
